# motor_task
此任务实现了电机的三角函数曲线速度规划，急动度渐变，纵享丝滑~

示例：
```
    arm2006_motor.posWithSpeedControl(360.f, 22.0f, 120.0f);
    // arm2006_motor电机转到360.0°处，最大速度为22.0个单位，缓冲位移为22.0°（包括初始缓冲与末端缓冲）
```

```
    arm2006_motor.posWithSpeedControl(360.f, 22.0f, 120.0f, 0.5f);
    // arm2006_motor电机转到360.0°处，最大速度为22.0个单位，缓冲位移为22.0°（包括初始缓冲与末端缓冲），初始转速为0.5个单位
```

```
    arm2006_motor.posWithSpeedControl(360.f, 22.0f, 120.0f, 0.5f, 0.0f);
    // arm2006_motor电机转到360.0°处，最大速度为22.0个单位，缓冲位移为22.0°（包括初始缓冲与末端缓冲），初始转速为0.5个单位，末端转速为0.0个单位
```

此外，MotorBase的getIsFinished()函数可以用来判断电机是否完成运动，返回值为bool类型，true表示完成，false表示未完成。
